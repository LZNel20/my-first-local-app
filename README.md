Hi! I hope you enjoy playing my adventure game. In my Main.py file is where you can find my project which is an adventure game about exploring different areas. When you run the code using python3 Main.py for mac, you simply will be asked your name, after it will ask you where you want to explore. All options will expect a, b, c, d, ect as responses, except for the eating menu, you will need to type out the name of what you want to eat. As you explore you will encounter varius things you can fight, hunt, or interact with, e.g. a deer. My project only uses the random inport as a dependance. Once you choose an area to explore you will be asked if you want to explore, check your inventory, eat some of your food, or switch areas. Switching areas may be disabled if you get yourself lost, but exploring more will help you find your way back.
